# Frontier historical concepts

このDirectoryは、改訂によって置き換えられたFrontierのSuperseded／Historical Conceptを保存します。現在案、正式仕様、実装指示または採用候補の正本ではなく、過去の判断理由を確認する場合だけ参照します。

新しいConceptや割り当て済みTaskから明示的に参照されない限り、旧版の記述を要求事項として取り込みません。旧版は原則として削除せずこのDirectoryへ移し、その際は親Directoryの[Index](../README.md)を更新します。
