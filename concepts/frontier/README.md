# Frontier concepts

Frontier全体、下位World群およびAdventure／Exploration Contentの非正本Conceptを索引します。ここに掲載された文書は正式な現行Frontier仕様ではなく、Plugin取得、World生成、Runtime変更またはRelease Scopeを承認しません。

## Current design input

- [Frontier_Server_Specification_V0.0.3.md](Frontier_Server_Specification_V0.0.3.md)
  - Frontier全体と、下位World群や共通境界を横断する上位Conceptです。
  - 正式な現行Frontier仕様ではありません。
- [Worlds_Beyond_Specification_V0.0.3.md](Worlds_Beyond_Specification_V0.0.3.md)
  - Iris Engineを使用する探索・観光Worldの構想です。
  - Iris Engineの導入、取得、World生成またはRelease Scopeを承認しません。
- [Ruined_Frontier_Specification_V0.0.3.md](Ruined_Frontier_Specification_V0.0.3.md)
  - BetterStructuresとEliteMobsを使用するWorld群の構想です。
  - BetterStructuresまたはEliteMobsの導入・取得を承認しません。

実装には、採用範囲を明示し、ユーザーが現在のSessionで割り当てた専用`codex/`指示書が必要です。
